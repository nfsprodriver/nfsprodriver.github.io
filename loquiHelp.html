"DECRYPT ERROR!"

In most cases you see this message the so called cipher keys could not be created correctly to decrypt that message, e.g. because existing ones could not be overwritten. Either you can wait a several time to get that message decrypted or you deleted your WhatsApp account e.g. using an Android client (Emulator ?) and recreate the account via LoquiIM. We're currently not able to solve this. Maybe we'll be lucky in the future. Thanks for understanding :)
